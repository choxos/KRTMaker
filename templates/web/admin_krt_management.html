Generation Commands

Use these Django management commands to generate admin KRTs:

Generate 10 random KRTs with Anthropic:
python manage.py generate_admin_krts --provider anthropic --model claude-3-5-sonnet-20241022 --limit 10 --random
Generate specific DOIs with Gemini:
python manage.py generate_admin_krts --provider gemini --model gemini-1.5-pro --dois "10.1101/2025.01.01.123456" "10.1101/2025.01.02.234567"
Tip: Use --dry-run to see what would be processed without actually generating KRTs. Add --auto-approve to automatically approve generated KRTs if you trust the model quality.
Generate Admin KRTs

Use the Django management command to generate admin KRTs. Copy and run one of the commands below in your terminal:

Generate 5 random papers (Anthropic Claude)

python manage.py generate_admin_krts --provider anthropic --limit 5 --random --auto-approve

Good for testing and getting started
Generate 10 papers (Gemini Pro)

python manage.py generate_admin_krts --provider gemini --model gemini-1.5-pro --limit 10

Use Google's Gemini model for generation
Dry run to see what would be processed

python manage.py generate_admin_krts --provider anthropic --limit 20 --dry-run

See what papers would be processed without generating KRTs
Note: You need to run these commands in your terminal/server where the Django project is deployed. Make sure you have activated your virtual environment and set the proper API keys.
